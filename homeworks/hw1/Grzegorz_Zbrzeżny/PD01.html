Identyfikacja zmiennej objaśnianej¶

Za pomoc tego data setu chcemy przewidzie, czy pacjent powrci do szpitala po zwolnieniu go z niego (etykieta readmitted). Moemy wic od razu wykluczy pewne kolumny, ktre to na pewno nie s z tym zwizane.

  • encounter_id - to unikalna warto dla kadego przyjia do szpitala, nie ma zwizku z jakimikolwiek czynnikami chorobowymi, mona wic usun te kolumn, by unikn potencjalnego przeuczenia
  • patient_nbr - jest indywidualna dla kadego pacjenta, wic zostawienie jej ponownie mogoby prowadzi do przeuczenia

Pozostae kolumny s ju mniej lub bardziej zwizane z czynnikami chorobowymi (przyjmowane substancje leczce, rodzaj diagnozy, itd.), wic mog decydowa o readmisji danego pacjenta. Dokadny opis poszczeglnych kolumn mona znale pod linkiem: https://www.hindawi.com/journals/bmri/2014/781670/tab1/ dlatego nie bd ju przepisywa tych opisw do raportu

Encoding zmiennych kategorycznych¶

Wpierw moemy encodowa kolumny z substancjami medycznymi przyjmowanymi przez pacjentw.

In [274]:
le = preprocessing.LabelEncoder()
le.fit(data["insulin"])
print(f"Wartoci w tych kolumnach to: {list(le.classes_)}")

Wartoci w tych kolumnach to: ['Down', 'No', 'Steady', 'Up']

Jako, e w tych kolumnach wystpuj jedynie 4 rne wartoci, w tym No oznacza, e pacjent w ogle nie przyjmuje danej substancji, natomiast pozostae wartoci to rne warianty przyjmowania (Down - przyjmuje mniej, Steady - przyjmuje tyle samo, Up - przyjmuje wicej). Moemy wic zastosowa kodowanie, gdzie 0 bdzie sytuacja, gdy pacjent w ogle nie przyjmuje leku, a 1 gdy przyjmuje.

In [275]:
dict = {"No" : 0, "Down" : 1, "Steady" : 1, "Up" : 1}
for i in range(19, 39):
    data.iloc[:, i] = data.iloc[:, i].map(dict)

Ponadto moemy to rwnie zrobi dla kolumn gender, change, oraz diabetesMed, gdy s to kolumny binarne.

In [276]:
names = ["gender", "change", "diabetesMed"]
for i in names:
    le2 = preprocessing.LabelEncoder()
    le2.fit(data[i])
    data[i] = le2.transform(data[i])

Dla kolumn race, max_glu_serum i A1Cresult wykonamy One Hot Encoding, gdy przyjmuj one 4, bd 5 rnych wartoci. Dodatkowo moemy zrobi one hot'a dla kolumny admission_type_id i num_procedures, gdy zawieraj one tylko 7 rnych wartoci.

Transformacje danych¶

Moemy przetransformowa niektre kolumny zawierajce wiele wartoci liczbowych poprzez zgrupowanie tych wartoci przedziaami. Zrobimy to po kolei na kolumnach time_in_hospital, number_outpatient

Nie potrzeba nam raczej informacji o dokadnej iloci dni tylko wystarczy, aby to bya oglna informacja o dugoci pobytu, pogrupujmy wic wartoci od 0 do 14 na grupy short - 0, medium - 1 i long - 2.

In [280]:
data["time_in_hospital"] = pd.cut(data["time_in_hospital"], bins = [0., 4, 9, np.inf], labels = [0, 1, 2])

Dla kolumn number_outpatient, number_emergency i number_inpatient wartoci powyej 10 wystpuj po naprawd niewiele razy w stosunku do tych od 0 do 10, wic moemy zastpi je wartoci 10.

In [281]:
data.loc[data["number_outpatient"] >= 10, "number_outpatient"] = 10
data.loc[data["number_emergency"] >= 10, "number_emergency"] = 10
data.loc[data["number_inpatient"] >= 10, "number_inpatient"] =10

Przeksztamy kolumn age na wartoci numeryczne. Skoro mamy podane przedziay wieku moemy bez straty oglnoci przeksztaci je na rodki tych przedziaw

In [282]:
def extract_and_get_mean(x):
    values = re.findall(r'\d+', x)
    return (int(values[0]) + int(values[1]))/2
In [283]:
data["age"] = data["age"].apply(extract_and_get_mean)

data = data.astype({"time_in_hospital" : int, "age" : int})

def group_diagnoses(x):
    if (x[0].lower() in alfabet or (280 < math.floor(float(x)) < 289) or (320 < math.floor(float(x)) < 329) or (630 < math.floor(float(x)) < 679) or (360 < math.floor(float(x)) < 389) or (740 < math.floor(float(x)) < 749)):
        return "Other"
    elif (390 < math.floor(float(x)) < 459 or math.floor(float(x)) == 785):
        return  "Circulatory"
    elif (460 < math.floor(float(x)) < 519 or math.floor(float(x)) == 786):
        return  "Respiratory"
    elif (520 < math.floor(float(x)) < 579 or math.floor(float(x)) == 787):
        return "Digestive"
    elif (800 < math.floor(float(x)) < 999):
        return "Injury"
    elif (710 < math.floor(float(x)) < 739):
        return "Musculoskeletal"
    elif (580 < math.floor(float(x)) < 629 or math.floor(float(x)) == 788):
        return  "Genitourinary"
    elif (math.floor(float(x)) == 250):
        return  "Diabetes"
    else:
        return "Neoplasms"
